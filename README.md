# Lu Renderer

A Tiny Renderer relying on minimum dependencies

## Content

- basic linear algebra
  - vector
  - quaternion
  - matrix
- Image IO
  - BMP Format
- Mesh IO
  - OBJ Format

## Compile

### Windows:

PowerShell: `mingw32-make`

Command Promot: `mingw32-make`

### MacOS:

ZSH: `make`

MacOS viewer window: `make mac`
